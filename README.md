# mpv-configuration
The configuration of my MPV media player, including some scripts written by other authors

## Windows平台中，MPV多媒体播放器的配置文件位于 %AppData%\mpv 路径下：
```
mpv
|-mpv.conf
|-input.conf
|-watch_later
  |-...
|-scripts[optional]
  |-...
|-lua-settings[optional]
  |-...
```
Notes: 本仓库中的内容仅为我个人根据使用习惯编写的设置，你完全可以自定义。
